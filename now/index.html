KTV Singer ktvsinger.com

An Apple TV karaoke app where the phone is the remote. Search a song, queue it, and the TV plays the YouTube video with synced lyrics from LRCLIB while streaming your mic back in real time. The stack is one of those “small-team monorepo that happens to span seven submodules” situations — Express + Socket.IO + Drizzle on the server, Expo React Native on the phone, SwiftUI on tvOS. April has been about making the internals trustworthy: reworked the mic pipeline to 100ms binary PCM chunks, passed a security review, and just shipped a set of Claude Code commands that encode the repo’s conventions so future work stays consistent. Next: vocal separation for singing over instrumentals of any YouTube track.
